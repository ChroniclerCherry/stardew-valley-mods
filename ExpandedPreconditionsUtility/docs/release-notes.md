# Release notes
## 1.1.2
Released 15 March 2026 for SMAPI 4.1.10 or later. Updated by Pathoschild.

- Expanded Preconditions Utility downloads are now created via [automated and attested builds](https://www.patreon.com/posts/automated-builds-148417912).  
  _This guarantees that the download only contains what's in the public source code and hasn't been tampered with._

## 1.1.1
Released 07 September 2025 for SMAPI 4.1.10 or later. Updated by Pathoschild.

- Fixed errors when the `config.json` has invalid values or the API is used incorrectly.
- Internal changes to simplify maintenance.

## 1.1.0
Released 21 January 2025 for SMAPI 4.1.10 or later. Updated by Pathoschild.

- Updated for Stardew Valley 1.6.15.
- You can now [check expanded preconditions using a game state query](README.md#usage).
